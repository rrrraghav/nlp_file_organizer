DESCRIPTION OF CHANGE:

This establishes cigarette and packaging specs for a new brand,
KENT SPECIAL MILD KS CHAR (JAPAN), Brand Code 1508, Product Code 5346
for Japan. Cigarette specs are identical to KENT MILDS KS CHAR (JAPAN)
except they use longer (30 mm) tipping.

New brand.
